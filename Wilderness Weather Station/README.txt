Installation Steps

1 – Extract and open the folder
 
2 – Copy and move the “template.xlxs” file into your computers local documents folder
 
3 – Once the template file has been moved the program is ready to run. 
To locate the .exe go to the extracted folder and then follow the path: \Wilderness Weather Station\bin\Debug

4 – Wilderness Weather Station.exe is located in the Debug folder and will be ready to run.
